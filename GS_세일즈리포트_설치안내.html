1 앱 설치 방법

🍎 iPhone (Safari)

①Safari에서 위 URL 접속
②하단 공유 버튼 (□↑) 탭
③"홈 화면에 추가" 선택
④이름 확인 후 "추가" 탭
⑤홈 화면에서 앱 아이콘으로 실행

🤖 Android (Chrome)

①Chrome에서 위 URL 접속
②"앱 설치" 배너 자동 표시
③또는 우측 상단 ⋮ → "앱 설치"
④홈 화면에 앱 아이콘 생성 완료
💡 설치 후: 앱을 열면 URL 바 없이 전체 화면으로 실행됩니다. 일반 앱처럼 사용하면 되며, 데이터는 서버 업데이트 시 자동 반영됩니다. 별도의 로그인이나 업데이트 작업이 필요 없습니다.
