인바운드 거래처 국적 매핑 관리

57-prefix 회원번호(구코드)는 회원명에 국적이 표기되지 않아 자동 매핑이 어렵습니다. 이 페이지에서 거래처별 국적을 수동 매핑하면 build_inbound_enriched.py 빌드 시 우선 적용됩니다.
우선순위: 회원명 (국적) 표기 > 키인 매핑 > 87 master > 미확인

🛠️ 사용법
  1. 아래 표에서 미확인 거래처의 국적을 드롭다운으로 선택 (또는 직접 입력)
  2. 「JSON 내보내기」 클릭 → inbound_partner_nationality_keyin.json 다운로드
  3. 다운로드한 파일을 docs/data/ 폴더에 덮어쓰기 → 커밋 & 푸시
  4. 다음 빌드(scripts/build_inbound_enriched.py)에서 자동 반영
