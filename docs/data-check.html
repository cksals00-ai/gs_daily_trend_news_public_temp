⚠️취소 반영 유의사항
예약일 기준 vs 투숙일 기준 차이 안내
• 예약일 기준: 코드 27(신규)+43(변경)-28(취소)-44(취소변경) — 취소 차감 포함
• 투숙일 기준: 코드 27(신규)+43(변경)만 집계 — 취소 미반영
→ PMS 원본과 대시보드 간 소폭 차이(1~3%)는 이 기준 차이로 발생할 수 있습니다.
→ 3% 이내 차이는 정상 범위로 간주합니다.
