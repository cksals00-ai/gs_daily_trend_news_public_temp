개요

FCST(Forecast) 키인은 각 사업장 담당자가 향후 매출 예측 데이터를 입력하는 프로세스입니다. 정확한 예측은 전사 경영계획 수립의 핵심 기초 데이터가 됩니다.

키인 절차

1FCST 키인 페이지 접속 — 상단 "FCST 키인" 카드 또는 fcst-admin.html로 접속
2사업장 선택 — 담당 사업장을 드롭다운에서 선택
3기간 설정 — 예측 대상 월을 선택 (당월+1 ~ +3개월)
4데이터 입력 — 객실 매출, F&B, 부대시설 등 항목별 예측값 입력
5저장 및 확인 — 입력 데이터 검증 후 저장

주의사항

  • 키인 마감일: 매월 25일까지 완료
  • 수정이 필요한 경우, 마감일 전까지 자유롭게 재입력 가능
  • 전년 동기 대비 ±30% 이상 차이 시 사유 코멘트 필수
  • 키인 현황은 fcst-status.html에서 실시간 확인 가능

개요

품의서는 가격 조정, 특별 할인, 프로모션 등 영업 관련 의사결정을 위한 결재 문서입니다. 정확한 양식 작성이 신속한 결재의 핵심입니다.

작성 절차

1품의 작성 페이지 접속 — 상단 "품의 작성" 카드 또는 pumui.html로 접속
2품의 유형 선택 — 가격조정 / 프로모션 / 특별요청 등 유형 선택
3기본 정보 입력 — 대상 사업장, 적용 기간, 대상 상품 입력
4상세 내용 작성 — 조정 내역, 사유, 예상 효과 기술
5첨부 및 생성 — 관련 자료 첨부 후 품의서 PDF 생성

작성 팁

  • 사유란에 경쟁사 동향, 시장 상황 등 객관적 근거를 포함하면 결재 속도 향상
  • 예상 매출 효과를 수치화하여 기재 권장
  • 긴급 건의 경우 제목에 [긴급] 표기
  • 이전 승인 사례 참고 시 품의번호를 함께 기재
